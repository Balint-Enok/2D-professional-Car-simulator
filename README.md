# 2D-professional-Car-simulator
Car Simulator v3.5 – A browser-based racing simulation with realistic physics, custom track editor, telemetry recording, and ghost car functionality. Features adjustable vehicle parameters (mass, tires, aero, transmission), torque curve editor, skin customization, and live telemetry comparison. Built with vanilla JS, Canvas, Web Audio.


## 🎯 Features

### 🏎️ Vehicle Physics
- **Realistic car model** with mass, inertia, center of gravity
- **Tire model** with slip angles, grip, and lateral forces
- **Engine simulation** with custom torque curve editor
- **Transmission** with manual/automatic modes and custom gear ratios
- **Aerodynamics** with adjustable downforce
- **Assistance systems**: ABS, Traction Control

### 🗺️ Track Editor
- **Custom track creation** with Catmull-Rom splines
- **Zoom/Pan** with mouse and touch support
- **Pinch-to-zoom** on mobile devices
- **Track width** adjustment
- **Import/Export** tracks as JSON

### 📊 Telemetry & Ghost Car
- **Real-time telemetry recording** (throttle, brake, steering, speed, RPM)
- **Best lap storage** with full data export
- **Ghost car** playback with comparison overlay
- **Imported telemetry** support for learning from others

### 🎨 Customization
- **Car shapes**: Race, F1, GT, Street
- **Colors**: Body, accent, windows, wheels, rims
- **Liveries**: Solid, gradient, racing stripes, two-tone, carbon, checkered
- **Number plate** with custom color

### 🎮 Controls
- **Keyboard**: W/A/S/D, Arrow keys, Space (handbrake), E/Q (gears)
- **Touch**: Steering wheel, gas/brake pedals, handbrake
- **Joystick** support with sensitivity adjustment
